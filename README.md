# 📚 Online Book Store Database Management System | PostgreSQL

## Project Overview
This project demonstrates the design and management of a relational database for an **Online Book Store** using **PostgreSQL**. It focuses on solving real-world business problems through SQL by performing data retrieval, filtering, aggregation, and analytical queries on bookstore data.

The project includes **20+ SQL queries** ranging from basic data extraction to advanced data analysis using **JOINs, GROUP BY, HAVING, Subqueries, Common Table Expressions (CTEs), and Window Functions**. These queries generate valuable business insights such as revenue analysis, inventory tracking, customer purchasing behavior, top-selling books, and sales trends.

This project showcases strong SQL querying, database management, and analytical problem-solving skills while applying PostgreSQL concepts to a practical business scenario.

## Key Features
- Database design and management using PostgreSQL
- Data retrieval and filtering
- Multi-table JOIN operations
- Aggregate functions and grouping
- Subqueries and Common Table Expressions (CTEs)
- Window Functions for analytical reporting
- Revenue and sales analysis
- Inventory management
- Customer purchase behavior analysis
- Best-selling books and order trend analysis
